Build a complete production-ready Telegram + Discord automation system using Python.

The project must be fully compatible with Railway deployment.

=========================
GENERAL REQUIREMENTS
=========================

- Python 3.12+
- Everything should run from a single file named main.py.
- Use aiogram for the Telegram bot.
- Use discord.py for the Discord bot.
- Fully asynchronous.
- Professional logging.
- Proper error handling.
- English language only.
- Clean and modern code.
- No placeholder code.
- Ready to run immediately.

Generate:

- main.py
- requirements.txt
- Procfile
- runtime.txt
- railway.json
- README.md
- .gitignore

The project must start with:

python main.py

=========================
ENVIRONMENT VARIABLES
=========================

Never hardcode tokens.

Use Railway Environment Variables.

Required:

TELEGRAM_BOT_TOKEN
DISCORD_BOT_TOKEN
OWNER_ID
